Ionic Seed
==========


## Requirements
- npm
- gulp

## Installation
```bash
# npm dependencies
npm install

# typings dependencies
gulp typings:install
```

## Serve app
```bash
gulp serve
```

## Server commands
```bash
# Serve server
gulp serve
```

## Build commands
```bash
# Webpack build
gulp webpack:build
```

## Typings commands
```bash
# Search module
gulp typings:search --module mymodule

# Install module
gulp typings:install --module mymodule [--global]
```
