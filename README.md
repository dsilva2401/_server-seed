Server Seed
==========


## Requirements
- npm
- gulp

## Installation
```bash
# npm dependencies
npm install

# typings dependencies
gulp typings:install
```

## Serve app
```bash
gulp serve
```

## Server commands
```bash
# Serve app
gulp serve
```

## Build commands
```bash
# Webpack build
gulp webpack:build
```

## Typings commands
```bash
# Search definition
gulp typings:search --module mymodule

# Install definition
gulp typings:install --module mymodule [--global]

# Uninnstall definition
gulp typings:uninstall --module mymodule [--global]
```
